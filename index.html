HTTP

The Hypertext Transfer protocol is an application layer protocol. Communication between client computers and web servers is done by sending HTTP Requests and receiving HTTP Responses.

HTML CSS

HTML (HyperText Markup Language) is the standard markup language for webpages, CSS is the language we use to style an HTML document. CSS describes how HTML documents should be displayed.

JAVASCRIPT

JavaScipt is the programming language of the Web. JavaScipt often abbreviated to JS is a Lightweight, interpreted, or just-in-time compiled programming language with firs-class functions.

GOLANG

Go language is a programming language initially developed at Google. It is statically-typed language having syntax to that of C. Go is used to create computer programs.
